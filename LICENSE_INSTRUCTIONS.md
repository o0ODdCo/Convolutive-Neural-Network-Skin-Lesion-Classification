# Choosing a software license

No license has been selected automatically.

Before making the repository public, choose the software license that matches your intent.

Typical options include:
- MIT: permissive reuse with attribution.
- Apache-2.0: permissive reuse with explicit patent terms.
- GPL-3.0: derivative works distributed under the same license conditions.

After choosing:
1. Add the official license text as a root-level file named `LICENSE`.
2. Replace or remove this instruction file.
3. Make sure dataset licensing is handled separately from your source-code license.

Do not assume that licensing your code gives you the right to redistribute the HAM10000 dataset.
